Learn How To build a Full Front-End Website With HTML CSS And JavaScript

By Aziz Boujaada & Online Education

Start this course and finish it step by step to mastering Front-End

In this 7-9-hour project, you'll create a website that showcases your qualifications, work experience,
and work samples, using HTML, CSS, JavaScript and upload your findings to a portfolio
page to showcase to potential employers.

Learning Objectives

  • Prove that you can integrate HTML, CSS, and JavaScript into a website
  • Showcase other works that you have already done and other online presence
  • Provide further information about your qualifications, references, and contact information

Part 1:Learn HTML

What is HTML

00:47 min

In this video, Hitesh introduces the basics of HTML (Hypertext Markup Language), one of the most essential elements of web
development. He explains the simple structure of an HTML document, which is divided into two main sections: the head and the body. The head contains metadata,
such as the document title, while the body contains visible content like headings (h1) and paragraphs.
Hitesh also discusses the use of VS Code and its Emmet abbreviation shortcuts to quickly generate HTML boilerplate code,
making development more efficient. He walks through the creation of a basic webpage, covering elements like the doctype,
which declares the version of HTML5, and the title and h1 tags. Additionally, Hitesh highlights the importance of writing semantic HTML
that works across various platforms, including screen readers and in-flight entertainment systems, ensuring accessibility and cross-device
compatibility. The video provides a strong foundation for understanding HTML structure and paves the way for exploring more advanced tags and web development techniques
in upcoming lessons. It's an excellent introduction for anyone starting with web development or frontend design.
